# defensa-asalto-base
Juego de estrategia por turnos en Python/Tkinter. Dos jugadores: uno defiende una base con torres y muros, el otro la ataca con unidades. Introducción a la Programación. Creadores: Julián Sanabria e Imanol Ovares



## Integrantes

- Julián Sanabria Loaiza
- Imanol Ovares

## Requisitos

- Python 3 (en el desarrollo del proyecto se usó python 3.14.3)
- Pillow
- Tkinter

## Instalación
En la terminal:
Clonar el github, entrar a la carpeta del programa e instalar pillow y tkinter
```bash
git clone https://github.com/JSL6767/defensa-asalto-base.git
cd defensa-asalto-base
pip install pillow
pip install tkinter
```

## Ejecución
opcion 1:
ejecutar en la terminal 
```bash
py main.py
```

opcion 2:
ejecutarlo normalmente, si estás en visual studio code

En sistemas Linux/Mac usar `python3 main.py`.
